## 工具设置

Skills 定义工具怎么用。这文件记你的具体情况 — 你独有的设置。

### 这里记什么

加上任何能帮你干活的东西。这是你的小抄。比如：

- SSH 主机和别名
- 其他执行 skills 时和用户相关的设置

**示例：** `home-server → 192.168.1.100，用户：admin`
